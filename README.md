# Python Automation Toolkit

A collection of beginner-friendly Python automation scripts focused on
digital organization and productivity.

## Current Tool

### File Organizer
Automatically organizes files inside a selected folder into categories
such as PDFs, Images, Videos, Applications, and Others.

**Features:**
- Uses `pathlib` for modern path handling
- Prevents overwriting existing files
- Clean folder structure with categorized directories
- Beginner-readable code structure

## Technologies Used
- Python 3
- Standard Library (`pathlib`)

## How to Use
1. Clone the repository
2. Run the script using Python
3. Follow the prompts to select a folder to organize
